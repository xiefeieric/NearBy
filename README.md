# NearBy
app基于英国几大华人公众号，比如英国红领巾，英国那些事儿。。。会让你很容易的找到附近的美食，并且这些美食是大家尝试过并且推荐的，点击进入相应的餐厅标记，就可以看到具体的图文并貌的餐厅食评，方便你选择。现阶段主要收录的是London的餐厅，我们会持续更新全英的餐厅。也欢迎大家发来餐厅评价的网页链接，我们也会添加进数据库，方便在英的华人。如果你也什么好的建议，也欢迎联系我改进。
